The official firethree. Create your own solona on-chain firestore, storage, analytics with this protocol.

## Quick Setup

### Install

Install these dependencies over:

npm:

```shell
npm install @firethreexyz/firethree-protocol
```

yarn:

```shell
yarn add @firethreexyz/firethree-protocol
```


### Build From Source

1. Clone the project:

```shell
git clone https://github.com/firethreexyz/firethree-protocol.git
```

2. Install dependencies:

```shell
cd firethree-protocol
npm install
```
